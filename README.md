# 買い物メモ (Shopping Memo)

シンプルな買い物リストアプリです。

## 機能

- ✅ アイテムの追加
- 📂 カテゴリ自動分類（食品/日用品/その他）
- ✔️ チェックで完了
- ↔️ スワイプで削除
- 💡 よく買うもの提案

## 技術スタック

- React Native with Expo
- Expo Router
- TypeScript
- AsyncStorage

## 開発

```bash
# インストール
npm install

# 開発サーバー起動
npm start

# テスト実行
npm test
```

## ビルド

```bash
# iOS
eas build --platform ios

# Android
eas build --platform android
```
